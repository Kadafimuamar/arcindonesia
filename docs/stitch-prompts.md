# Google Stitch Master Prompt

Gunakan prompt berikut sebagai basis desain Google Stitch untuk seluruh varian halaman RangkumanKu:

## Master Prompt
Buat web app editorial blog modern berbahasa Indonesia bernama RangkumanKu. Gunakan warm off-white background, dark navy header dan footer, orange accent color, rounded corner cards, shadow lembut, spacing system 8px, typography sans-serif yang mudah dibaca, dan judul artikel bergaya editorial. Pastikan accessible color contrast, keyboard navigation, focus state terlihat, serta responsif untuk mobile, tablet, dan desktop. Buat state loading, empty, error, dan confirmation. Tabel pada mobile harus berubah menjadi card stack. Sertakan versi desktop dan mobile.

## Daftar layar
1. Home page dengan featured article, search bar, latest posts, pagination, mobile navigation.
2. Semua tulisan dengan grid artikel, sorting terbaru/terlama, shortcut ke pencarian.
3. Detail tulisan dengan breadcrumb, metadata penulis, markdown content, share actions, related articles, comments.
4. Hasil pencarian dengan search term aktif, empty state, pagination.
5. Login admin dengan form email/password, toggle show password, loading dan error state.
6. Dashboard admin dengan statistik, recent posts, recent comments, sidebar responsif.
7. Daftar tulisan admin dengan search, filter status, action buttons, confirmation dialog.
8. Tambah/edit tulisan dengan markdown editor, live preview, upload cover, dirty state warning.
9. Kelola komentar dengan filter, search, hide/restore/delete actions, admin-only email visibility.
10. State loading, empty, error, confirmation untuk setiap layar kunci.
